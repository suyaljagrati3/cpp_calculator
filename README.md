# C++ Calculator

A simple command-line calculator written in C++ that performs addition, subtraction, multiplication, and division on multiple numbers.

## Features
- Supports +, -, *, / operators
- Works with multiple numbers at once
- Prevents division by zero
- Interactive loop for multiple calculations

## How to Run

1. Make sure you have a C++ compiler installed (e.g., g++).

2. Open a terminal in the folder containing `calculator.cc`.

3. Compile the program:
   ```bash
   g++ calculator.cc -o calculator   # Creates an executable named 'calculator'
